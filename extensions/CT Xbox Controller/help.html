Xbox Controller

What is it?

Everyone knows what an Xbox Controller is, especially true "Gamers".

****MORE STUFF YOU HAVE TO DO****

When adding an XboxController to the project, the imports you need aren't done automatically.

You need to cut and paste the following three imports into the OI.java file.

import edu.wpi.first.wpilibj.XboxController;

import edu.wpi.first.wpilibj.GenericHID.Hand;

import edu.wpi.first.wpilibj.GenericHID.RumbleType;

Properties

PORT
The USB Port used to communicate with the Xbox Controller.
